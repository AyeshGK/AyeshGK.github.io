Clinical Trial Management System

For the Object-Oriented Software Development module, we developed a website for Clinical Trial Management.This is a Web application built using PHP, MySQL with MVC for backend and HTML, CSS, Bootstrap for Frontend.

Tech Stack

  • PHP
  • MySQL database with Prisma ORM
  • JavaScript
  • Bootstrap
  • HTML&CSS

Functionalities

    Implemented by using PHP and MySQL
  • A company can add/edit/delete clinical trials.
  • A client can apply and participate in clinical trials.
  • Developed a web application for pharmaceutical companies to manage clinical trials.
  • Provides a common platform for both companies and clients to meet.

Implementation

  • MVC Architecture
  • SMTP Email support
